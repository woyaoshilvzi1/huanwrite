# 网文选题筛选规则

## 判断目标

- [ ] 一句话能说清主角、压力、行动和读者期待。
- [ ] 卖点能在开头快速显形，不依赖长背景解释。
- [ ] 冲突能持续升级，不是只靠误会拖延。
- [ ] 主角有主动选择，不只是被剧情推着走。
- [ ] 目标平台、目标体量和题材承诺一致。

## 优先题材信号

- [ ] 弹幕、系统、重生、穿书等外挂信息能改变主角当下行动。
- [ ] 真假千金、家庭压迫、婚恋关系、校园关系等冲突能落到具体利益。
- [ ] 身份反差能立刻转成行动压力。
- [ ] 关系破裂、重逢修罗场、危机压身能带来明确后果。

## 淘汰信号

- [ ] 只有设定，没有事件。
- [ ] 只有情绪，没有行动。
- [ ] 只有人设标签，没有关系变化。
- [ ] 只能靠解释背景才能理解卖点。
