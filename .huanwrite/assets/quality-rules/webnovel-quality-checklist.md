# 网文质量检查清单

## 章内结构

- [ ] 本章开头有抓手，不平铺日常。
- [ ] 本章中段推进剧情、关系或设定理解。
- [ ] 本章后段局面已经和开头不同。
- [ ] 章末停在变化发生的一拍。

## 黄金三章

- [ ] 第 1 章有明确事件。
- [ ] 第 2 章冲突升级或卖点展开。
- [ ] 第 3 章建立继续追读的理由。

## 信息投放

- [ ] 没有大段背景直灌。
- [ ] 信息跟着冲突出现。
- [ ] 读者不需要一次记住过多设定。

## 语言

- [ ] 没有空泛抒情堆砌。
- [ ] 没有连续重复同一种情绪。
- [ ] 对话符合人物身份。
- [ ] 关键情绪有动作、反应或对白承载。
