# 语料资产说明

当前项目只接收可合法开源的语料摘要、标签、结构索引和自有样本。

未确认授权的完整小说正文、docx、pdf 不进入当前 MIT 开源仓库。需要使用外部语料时，先导入为本地私有资料，再由工具生成摘要、标签和结构索引；正式运行只读取当前项目 `data/assets/corpus/` 下的索引资产。

## 可用索引类型

- `article_profiles`：文章级摘要、题材标签、结构标签。
- `excerpts`：短摘录与结构标签。
- `imitation_index`：用于拆结构、拆节奏、拆信息投放的索引。

## 使用边界

- [ ] 只模仿结构、节奏、信息投放和章末截断。
- [ ] 不照抄原句。
- [ ] 不复制特定作者的表面文风。
- [ ] 不把多个范本硬拼成拼贴文本。
