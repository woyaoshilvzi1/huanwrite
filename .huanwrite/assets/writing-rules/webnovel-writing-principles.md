# 网文写作规则

## 开头

- [ ] 前几段必须有异常局面、压力或明确诱因。
- [ ] 主角尽快出场，并在压力中做出选择。
- [ ] 卖点和冲突绑定，不把卖点写成设定说明。
- [ ] 背景信息跟着动作、对白或冲突出现。

## 场景

- [ ] 每场戏有目标、阻碍和变化。
- [ ] 场景里有人在做事，不原地总结感受。
- [ ] 对白要改变关系、暴露利益或推进事件。
- [ ] 每段都应提供新信息、新动作或新压力。

## 章末

- [ ] 停在决定、发现、危险、反转或关系变化上。
- [ ] 不用总结式收尾。
- [ ] 不把下一章该兑现的信息提前说满。

## 去 AI 味

- [ ] 少用空泛情绪总结。
- [ ] 少用套话式形容词堆叠。
- [ ] 关键情绪用动作、反应、对白落地。
- [ ] 不把“去 AI 味”理解成故意粗糙。
