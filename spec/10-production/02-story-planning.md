# Story Planning Checklist

## 目标

定义正文生产前必须完成的写前规划。

## 规划内容

- [ ] 故事承诺已定义。
- [ ] 主角目标已定义。
- [ ] 主冲突已定义。
- [ ] 阻力来源已定义。
- [ ] 利益链已定义。
- [ ] 人物关系已定义。
- [ ] 章节或段落结构已定义。
- [ ] 每个章节或段落有目标。
- [ ] 每个章节或段落有冲突。
- [ ] 每个章节或段落有变化。
- [ ] 每个章节或段落有钩子或承接点。
- [ ] 文风规则已定义。
- [ ] 禁用句式已定义。
- [ ] 疲劳词已定义。
- [ ] 目标平台和稿件形态已定义。

## 规划状态

- [ ] 规划可以是草稿。
- [ ] 规划可以是待确认。
- [ ] 规划可以是已确认。
- [ ] 已确认规划修改后必须重新确认。

## 流程规则

- [ ] 规划未确认前不能进入正文候选生产。
- [ ] 正文候选必须能追溯到当时有效规划。
- [ ] 规划修改后，旧候选需要重新判断是否仍可用。

## 完成验收

- [ ] 每篇正式生产稿都有规划。
- [ ] 每个规划有确认状态。
- [ ] 每个正文候选能追溯规划。
