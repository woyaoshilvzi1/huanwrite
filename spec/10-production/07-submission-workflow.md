# Submission Workflow Checklist

## 目标

定义投稿包、投递记录和变现记录如何管理。

## 投稿包

- [ ] 投稿包关联正式稿件。
- [ ] 投稿包包含标题方案。
- [ ] 投稿包包含短简介。
- [ ] 投稿包包含长简介。
- [ ] 投稿包包含标签。
- [ ] 投稿包包含目标平台。
- [ ] 投稿包包含平台备注。
- [ ] 投稿包包含投前检查项。

## 平台规则

- [ ] 平台规则只记录公开信息。
- [ ] 平台规则有来源。
- [ ] 平台规则有核验时间。
- [ ] 平台规则有待确认项。
- [ ] 投前需要重新核验平台规则。

## 投递记录

- [ ] 投递记录关联投稿包。
- [ ] 投递记录有平台。
- [ ] 投递记录有投递时间。
- [ ] 投递记录有当前状态。
- [ ] 投递记录有下一步动作。
- [ ] 投递记录不保存账号密码或后台隐私。

## 收入记录

- [ ] 收入记录只记录已确认事实。
- [ ] 收入记录关联稿件或投稿记录。
- [ ] 收入记录区分保底、买断、分成、回款。
- [ ] 收入记录不保存银行卡、身份证、税务隐私。

## 完成验收

- [ ] 投稿包和投递记录分离。
- [ ] 平台规则和真实投递分离。
- [ ] 收入事实不提前虚构。
