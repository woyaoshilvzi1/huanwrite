# Topic Selection Checklist

## 目标

定义题材如何进入小说生产流程。

## 题材信息

- [ ] 每个题材有一句话说明。
- [ ] 每个题材有核心卖点。
- [ ] 每个题材有主角困境。
- [ ] 每个题材有主冲突。
- [ ] 每个题材有读者爽点。
- [ ] 每个题材有目标平台判断。
- [ ] 每个题材有目标体量判断。
- [ ] 每个题材有风险备注。

## 筛选标准

- [ ] 题材能在一句话里讲清。
- [ ] 开头能快速出现异常、损失、压迫或关系断裂。
- [ ] 主角有明确主动选择。
- [ ] 冲突连接钱、身份、资源、亲密关系、舆论或制度。
- [ ] 题材能支撑目标体量。
- [ ] 题材能对应至少一个投稿方向。

## 输出

- [ ] 题材可以进入规划。
- [ ] 题材可以暂停。
- [ ] 题材可以废弃。
- [ ] 题材判断有时间和理由记录。

## 完成验收

- [ ] 任一进入规划的题材都有完整判断。
- [ ] 题材状态可追踪。
- [ ] 题材不会直接跳过规划进入正文。
