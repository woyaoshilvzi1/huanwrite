# Candidate Workflow Checklist

## 目标

定义自动生成内容如何进入待审流程。

## 候选来源

- [ ] 续写可以生成候选。
- [ ] 改写可以生成候选。
- [ ] 去 AI 味可以生成候选。
- [ ] 按审稿返修可以生成候选。
- [ ] 投稿包生成可以生成候选。
- [ ] 短剧改编可以生成候选。

## 候选记录

- [ ] 候选有所属稿件。
- [ ] 候选有动作类型。
- [ ] 候选有生成时间。
- [ ] 候选有输入摘要。
- [ ] 候选有输出内容或输出路径。
- [ ] 候选有状态。
- [ ] 候选能追溯规划。

## 候选状态

- [ ] 候选可以是待审。
- [ ] 候选可以是审稿通过。
- [ ] 候选可以是审稿退回。
- [ ] 候选可以是已合入。
- [ ] 候选可以是废弃。

## 完成验收

- [ ] 任一自动生成结果都先进入候选。
- [ ] 候选不能绕过审稿进入正文。
- [ ] 候选状态可追踪。
