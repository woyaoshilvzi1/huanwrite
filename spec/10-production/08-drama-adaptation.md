# Drama Adaptation Checklist

## 目标

定义短篇小说如何改成短剧稿。

## 改编前提

- [ ] 原稿有正式正文。
- [ ] 原稿已经完成基础审稿。
- [ ] 原稿有明确主冲突。
- [ ] 原稿有可拆分的反转和钩子。
- [ ] 原稿适合短剧平台。

## 改编内容

- [ ] 改编稿关联原稿。
- [ ] 改编稿包含人物小传。
- [ ] 改编稿包含 10 集一卡。
- [ ] 每集有开场冲突。
- [ ] 每集有人物选择。
- [ ] 每集有信息翻转。
- [ ] 每集有结尾钩子。
- [ ] 改编稿有平台备注。

## 状态

- [ ] 改编可以是候选。
- [ ] 改编可以是待审。
- [ ] 改编可以是已确认。
- [ ] 改编可以进入投稿包。

## 完成验收

- [ ] 短剧改编不覆盖原稿。
- [ ] 短剧改编有独立状态。
- [ ] 短剧改编能追溯原稿。
