# Review And Quality Checklist

## 目标

定义候选内容如何审稿和判断质量。

## 审稿依据

- [ ] 审稿读取当前稿件事实。
- [ ] 审稿读取当前有效规划。
- [ ] 审稿读取候选内容。
- [ ] 审稿读取目标平台要求。
- [ ] 审稿读取文风规则。
- [ ] 审稿读取禁用句式和疲劳词。

## 审稿内容

- [ ] 检查候选是否承接规划。
- [ ] 检查人物关系是否一致。
- [ ] 检查主角是否有主动性。
- [ ] 检查冲突是否连接利益链。
- [ ] 检查场景是否有变化。
- [ ] 检查对白是否有身份、利益和压迫。
- [ ] 检查证据物件是否具体。
- [ ] 检查章末或段落结尾是否有承接力。
- [ ] 检查 AI 味风险。
- [ ] 检查平台适配风险。

## 审稿结论

- [ ] 可以通过。
- [ ] 可以退回。
- [ ] 可以要求局部返修。
- [ ] 可以要求人工合入。
- [ ] 可以废弃候选。

## 完成验收

- [ ] 每个审稿结论有理由。
- [ ] 每个退回候选有明确修改方向。
- [ ] 审稿不直接修改正文。
