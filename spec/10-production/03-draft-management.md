# Draft Management Checklist

## 目标

定义正式正文如何保存、修改和确认。

## 正文职责

- [ ] 正文只保存人工确认后的稿件内容。
- [ ] 正文不保存提示词。
- [ ] 正文不保存运行日志。
- [ ] 正文不保存审稿意见。
- [ ] 正文不保存平台规则。
- [ ] 正文不保存候选池内容。

## 保存规则

- [ ] 正文保存有时间记录。
- [ ] 正文保存有来源记录。
- [ ] 正文保存能区分人工编辑和候选合入。
- [ ] 空内容不能覆盖已有正文。
- [ ] 候选不能自动覆盖正文。

## 合入规则

- [ ] 候选通过审稿后才能进入合入流程。
- [ ] 合入需要人工确认。
- [ ] 合入后正文状态更新。
- [ ] 合入后保留候选来源。

## 完成验收

- [ ] 正文和候选分离。
- [ ] 正文和审稿意见分离。
- [ ] 正文变更可追踪。
