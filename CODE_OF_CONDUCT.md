# Code of Conduct

## Standard

Contributors are expected to keep collaboration direct, factual, and respectful.

## Unacceptable Behavior

- Harassment or personal attacks.
- Publishing private information.
- Introducing secrets or private platform data into the repository.
- Deliberately bypassing project specs, tests, or security boundaries.

## Enforcement

The project maintainer may reject, revert, or block contributions that violate this code or the project rules.
